# Fantasy Draft ⚽

A real-time multiplayer fantasy soccer drafting application with trivia-based draft order. Built with React, TypeScript, Supabase, and Tailwind CSS.

## 🎮 Features

- **Real-time Multiplayer**: Join rooms with friends and draft together in real-time
- **Trivia-Based Draft Order**: Answer trivia questions to determine your draft position
- **Live Draft Room**: Competitive drafting interface with countdown timers
- **Player Pool Management**: Filter and search through hundreds of soccer players by position (GK, DEF, MID, FWD)
- **Team Board**: Track your drafted players and monitor other teams
- **Authentication**: Secure user authentication powered by Supabase
- **Responsive Design**: Beautiful glassmorphism UI that works on all devices
- **Sound Effects**: Immersive audio feedback for draft actions

## 🏗️ Tech Stack

### Frontend
- **React 18.3** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **React Router** - Client-side routing
- **Framer Motion** - Animations and motion effects
- **Tailwind CSS** - Utility-first styling
- **Shadcn/ui** - Beautiful UI components
- **Radix UI** - Accessible component primitives
- **TanStack Query** - Data fetching and state management
- **Zod** - Schema validation

### Backend
- **Supabase** - PostgreSQL database, authentication, and real-time subscriptions
- **Row Level Security (RLS)** - Database security policies

### Testing
- **Vitest** - Unit testing framework
- **Playwright** - End-to-end testing
- **React Testing Library** - Component testing utilities
